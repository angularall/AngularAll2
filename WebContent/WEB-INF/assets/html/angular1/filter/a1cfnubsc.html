Angular filter(|number)

Formats a number as text.

If the input is null or undefined, it will just be returned. If the input is infinite (Infinity or -Infinity), the Infinity symbol '∞' or '-∞' is returned, respectively. If the input is not a number an empty string is returned

For example to display a number value in comma separated text in html

var amount=500840.2350 /* its a number,and output in comma separated text with fraction */

<div>Account Balance: { {amount|number:2}}</div>

Output:

Account Balance:5,00,840.23

Usage

Html

{{ number_expression | number : fractionSize}}

JavaScript

$filter('number')(number, fractionSize)
