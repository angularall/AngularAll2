Angular filter(|currency)

Formats a number as a currency (i.e 1000 to $1,000.00)
When no currency symbol is provided, default symbol for current locale is used

For example to display account balance with currency format

var AccountBalance=1000;

<div>Balance:{ {AccountBalance|currency}}</div>
Output:
Balance:$1,000.00

Usage

In Javascript

$filter('currency')(amount, symbol, fractionSize)

In Html

{{ currency_expression | currency : symbol : fractionSize}}
