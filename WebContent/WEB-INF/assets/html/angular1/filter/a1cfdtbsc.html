Angular filter(|Date)

Formats a (date/number) as a Date (i.e 1482853596873 to "Dec 27, 2016")

For example to display the date of birth in html

var DOB=new Date(); /* Date object */

var DOB="1482853596873"; /* time in milliseconds as string */

var DOB=1482853596873; /* time in milliseconds as number */

<div>Date of Birth:{ {DOB|date}}</div>
Output:
Date of Birth:Dec 27, 2016

Usage

JavaScript

$filter('date')(date, format, timezone)

Html

{{ date_expression | date : format : timezone}}
