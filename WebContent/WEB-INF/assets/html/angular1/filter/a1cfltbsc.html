Angular filter(|limitTo)

Converts string to uppercase.

For example to display name in upper case in html

var name="Sachin Tendulkar" /* name has some caps in value */

<div>user name: { {name|uppercase}}</div>
Output:
user name:SACHIN TENDULKAR

Usage

Html

{{ limitTo_expression | limitTo : limit : begin}}

JavaScript

$filter('limitTo')(input, limit, begin)
