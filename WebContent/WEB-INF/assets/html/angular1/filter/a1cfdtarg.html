Arguments (|Date)

JavaScript

$filter('date')(date, format, timezone)

Html

{{ date_expression | date : format : timezone}}

Arguments

Parameter Data-Type Details
date number Date to format either as Date object, milliseconds (string or number) or various ISO 8601 datetime string formats
format(optional) string Formatting rules If not specified, mediumDate is used.
timezone(optional) string Timezone to be used for formatting, use a time zone offset, for example, '+0430'
returns string Formatted string or the input if input is not recognized as date/millis.
