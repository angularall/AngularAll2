Angular filter(|json)

Allows you to convert a JavaScript object into JSON string

For example to display entire object return from back end system like java and to see the data from that object

var mobileObject={'mobileName':"Apple",'price':65000,'model':"6s"} /* Its javascript object has some properties and values */

<div>Data is : { {mobileObject|json}}</div>

Output:

Data is : {"mobileName":"Apple","price":65000,"model":"6s"}

Usage

Html

{{ json_expression | json : spacing}}

JavaScript

$filter('json')(object, spacing)
