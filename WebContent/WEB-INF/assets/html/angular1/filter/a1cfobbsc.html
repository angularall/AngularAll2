Angular filter(|orderBy)

Returns an array containing the items from the specified collection, ordered by a comparator function based on the values computed using the expression predicate

For example to display country names in alphabetical order

var countryList=['USA','China','India','Japan','England','Canada']

<ul> <li ng-repeat="country in countryList|orderBy"> { {country}} </li> </ul>

Output:

Canada
China
England
India
Japan
USA

Usage

Html

{{ orderBy_expression | orderBy : expression : reverse : comparator}}

JavaScript

$filter('orderBy')(collection, expression, reverse, comparator)
