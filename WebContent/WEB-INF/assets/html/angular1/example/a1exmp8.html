Example-8: Display complex objects in table by using ng-repeat directive(usage of ng-repeat)

  • {{tab}} 

<html>
	<head>
		<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.1/angular.min.js"></script> <!-- Step-1 -->
		<script>
			angular.module('myApp', []); <!--Step-2 -->
			angular.module('myApp').controller('mobileController',function($scope){ <!-- Step-3 -->
				$scope.mobileData = [{'brand':'Apple','price':'800','model':'7s'},
				                     {'brand':'Samsung','price':'650','model':'SJ4'},
				                     {'brand':'Blackberry','price':'750','model':'Dtek'}];
			});
		</script>
	</head>
	<body ng-app="myApp"><!-- Step-4 -->
		<div  ng-controller="mobileController"> <!-- Step-5 -->
			<table>
      			<tr>
        			<th>Brand</th>
        			<th>Price</th>
        			<th>Model</th>
      			</tr>
      				<tr ng-repeat="(index,mobile) in mobileData"><!-- Step-6 -->
        				<td>{ {mobile.brand}}</td>
        				<td>{ {mobile.price}}</td>
        				<td>{ {mobile.model}}</td>
      				</tr>
    		</table>
		</div>
	</body>
</html>

Quidance

New concepts are below: goto All Steps for complete info
  • Complex array or array list objects are manipulated in html by using ng-repat and by using .(dot) notation to fetch data from iterated entity
  • ng-repeat with indexing: index of array element can be get by using (index,item) syntax and first param is index and second param is array element
  • If we are using ng-repeat="item in list" gives item alone but we can not use ng-repat="(item) in list" to get item
