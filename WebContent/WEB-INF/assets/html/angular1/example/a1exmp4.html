Example-3: Update mobile number by using angular ng-click directive(usage of ng-click)

  • {{tab}} 

<html>
	<head>
		<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.1/angular.min.js"></script> <!-- Step-1 -->
		<script>
			angular.module('myApp', []); <!--Step-2 -->
			angular.module('myApp').controller('mobileController',function($scope){ <!-- Step-3 -->
				$scope.mobileNum=9999999;
				$scope.myLatestNum=$scope.mobileNum;
				$scope.updateDetail=function(){   <!--Step-4-->
					$scope.myLatestNum=$scope.mobileNum;
				}
			});
		</script>
	</head>
	<body ng-app="myApp"><!-- Step-5 -->
		<div  ng-controller="mobileController"> <!-- Step-6 -->
			<p>Your mobile number:{{myLatestNum}}</p>
			<p>Update your number</p>
			<input type="number" ng-model="mobileNum"/><br>
			<input type="button" ng-click="updateDetail()" value="Update"/><!--Step-7-->
		</div>
	</body>
</html>

Quidance

New concepts are below: goto All Steps for complete info
  • 4.Define scope function for ng-click event inside controller function for event handling
  • 7.ng-click is used to bind event handler function in html
  • On click on update button$scope.updateDetail() function in mobileController will be executed and myLatestNum variable is updated with new value
