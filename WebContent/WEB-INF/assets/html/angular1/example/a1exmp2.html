<html>
	<head>
		<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.1/angular.min.js"></script> <!-- Step-1 -->
		<script>
			angular.module('myApp', []);<!--Step-2-->
		</script>
	</head>
	<body ng-app="myApp"><!-- Step-3 -->
		<div  ng-init="location='Enter'"> <!-- Step-4 -->
			Enter your birth place: <input type=text ng-model="Enter"/><!-- Step-5 -->
			<p>My Birth Country: { {location}} </p> <!-- Step-6 -->
		</div>
	</body>
</html>

Quidance

New concepts are below: goto All Steps for complete info
  • 4.Initialize location(Optional) variable with some text by using ng-init
  • 5.Bind text filed by using ng-model with the model variable(location)
  • 6.Print that variable { {location}} by using { { }} double curly braces
  • ng-model is and angular directive used in html tags to bind user data from text filed
  • In more simple ng-model is used to collect data from user and save it in javascrit variable
