Example-3: Prints fullName dynamically by using angular controller directive(usage of ng-controller and $scope)

  • {{tab}} 

<html>
	<head>
		<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.1/angular.min.js"></script> <!-- Step-1 -->
		<script>
			//define module
			angular.module('myApp', []); <!--Step-2 -->
			//define controller <!-- Step-3 -->
			angular.module('myApp').controller('firstController',function($scope){
				$scope.firstName="Albert";
				$scope.secondName="Einstein";
			});
		</script>
	</head>
	<body ng-app="myApp"><!-- Step-4 -->
		<div  ng-controller="firstController"> <!-- Step-5 -->
			<p>FirstName:<input type="text" ng-model="firstName"/></p>
			<p>SeconddName:<input type="text" ng-model="secondName"/></p>
			FullName={ {firstName+" "+secondName}} !
		</div>
	</body>
</html>

Quidance

New concepts are below: goto All Steps for complete info
  • 3.Define angular controller by calling angular.module("moduleName").controller() function
  • 5.Declare ng-controller in html tag for controller functioning area
  • $scope module is provided by angular for all controller functions and it holds all properties of a controller
  • controller models/properties in Script(firstName,lastName) are accessed by using $scope ($scope.firstName)
  • controller models/properties in Html(firstName,lastName) are accessed by using just name(ng-model="firstName") within angular directive(ng-model,ng-init,etc) and expression { {firstName}}
