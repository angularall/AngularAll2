Example1:Hello World!



<html>
	<head>
		<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.1/angular.min.js"></script>
	</head>
	<body>
		<div ng-app="myApp" ng-init="myText='Hello World!'">
			{{myText} }
		</div>
		<script>
			angular.module('myApp', []);
		</script>
	</body>
</html>

Quidance

  • 1.Create a html file then include the angular js lib file form CDN(content delivery network) or download and point from your directory
  • 2.Create angular module by calling angular.module function and specify first param as name in string and second param as [] empty array(will discuss the second param later)
  • 3.Specify the module area where the angular logic will execute i.e ng-app="myApp" in div
  • 4.Initialize some text in myText variable by using ng-init
