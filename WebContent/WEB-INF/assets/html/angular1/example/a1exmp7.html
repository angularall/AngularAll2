Example-7: Removing entries from list by using angular ng-click and looping with ng-repeat directive(usage of ng-click,ng-repeat)

  • {{tab}} 

<html>
	<head>
		<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.1/angular.min.js"></script> <!-- Step-1 -->
		<script>
			angular.module('myApp', []); <!--Step-2 -->
			angular.module('myApp').controller('mobileController',function($scope){ <!-- Step-3 -->
				$scope.mobileBrandList = ['Apple', 'Samsung', 'Nexes','BlackBerry','HTC'];
				$scope.newBrand="";
				$scope.addBrand = function() {  <!-- Step-4 -->
					$scope.mobileBrandList.push($scope.newBrand);
				};
				$scope.remove= function(index) {  <!-- Step-5 -->
					$scope.mobileBrandList.splice(index,1);
				};
			});
		</script>
	</head>
	<body ng-app="myApp"><!-- Step-6 -->
		<div  ng-controller="mobileController"> <!-- Step-7 -->
			<ul>
				<li ng-repeat="(index,mobileBrand) in mobileBrandList"><!-- Step-8 -->
					{{index}}.{{mobileBrand}} <a ng-click="remove(index)"> remove</a>
				</li>
			</ul>
			<br>
			<input type="text" ng-model="newBrand"/>
			<input type="button" ng-click="addBrand()" value="Add"/><!--Step-9-->
		</div>
	</body>
</html>

Quidance

New concepts are below: goto All Steps for complete info
  • ng-repeat with indexing: index of array element can be get by using (index,item) syntax and first param is index and second param is array element
  • if we use simple ng-repeat="item in list" gives item alone but we can not use ng-repat="(item) in list"
  • On click of remove link triggers click event and calls remove() function in controller
