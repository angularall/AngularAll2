Example-9: Display records in table and sorting by column (static)(usage of orderBy)

  • {{tab}} 

<html>
	<head>
		<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.1/angular.min.js"></script> <!-- Step-1 -->
		<script>
			angular.module('myApp', []); <!--Step-2 -->
			angular.module('myApp').controller('mobileController',function($scope){ <!-- Step-3 -->
				$scope.mobileData = [{'brand':'Apple','price':'800','model':'7s'},
				                     {'brand':'Samsung','price':'650','model':'SJ4'},
				                     {'brand':'Blackberry','price':'750','model':'Dtek'}];
			});
		</script>
	</head>
	<body ng-app="myApp"><!-- Step-4 -->
		<div  ng-controller="mobileController"> <!-- Step-5 -->
			<table>
      			<tr>
        			<th>Brand</th>
        			<th>Price</th>
        			<th>Model</th>
      			</tr>
      				<tr ng-repeat="(index,mobile) in mobileData|orderBy:'brand'"><!-- Step-6 -->
        				<td>{ {mobile.brand}}</td>
        				<td>{ {mobile.price}}</td>
        				<td>{ {mobile.model}}</td>
      				</tr>
    		</table>
		</div>
	</body>
</html>

Quidance

New concepts are below: goto All Steps for complete info
  • Step-6:orderBy is an angular directive used with ng-repeat to display records in sorting order
  • |orderBy:'propertyName' is used to display the records in alphabetical order
  • Reverse sorting can be achieved by simply adding minus(-) sign with property name which is used in orderBy statement
  • orderBy:'-brand' prints records in reverse order by brand name
