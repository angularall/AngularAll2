<html>
	<head>
		<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.1/angular.min.js"></script> <!-- Step-1 -->
		<script>
			angular.module('myApp', []);<!--Step-2-->
		</script>
	</head>
	<body ng-app="myApp"><!-- Step-3 -->
		<div  ng-init="myText='Hello World!'"> <!-- Step-4 -->
			{{myText} } <!-- Step-5 -->
		</div>
	</body>
</html>

Quidance

  • 1.Add angular js lib file(Mandatory) form CDN(content delivery network) or download and point from your file directory
  • 2.Define angular module(Optional) in script tag by calling angule.module().Provide app name as first param in string and second param as empty array(will discuss about the second param later)
  • 3.Define ng-app="myApp"(mandatory) in body tag and it is the starting point for angular execution in html and inside this module area the angular code logic will run
  • 4.Initialize myText(Optional) variable with some text by using ng-init
  • 5.Print variable { {myText}} by using { { }} double curly braces
  • if no angular.module("myApp")is written, then mention empty name in ng-app=" " in html(i.e ng-app with empty string for bootstrap/starting point)
  • angular.module("yourappname")is used to assign module name for your application and this is needed for angular library to add dependent module in your app(check angular module for more info)
  • ng-init is a angular directive used in angular to initialize model variables in html itself
  • Angular expression { { }}is an angular expression used to print variable text in html

  • ng-

    is the prefix for all angular directives in html(ex:ng-app,ng-init,..)
