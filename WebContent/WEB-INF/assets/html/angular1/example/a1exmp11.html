Example-11: Dynamic sorting for both ASC/DSC(usage of orderBy)

Relax:This is very simple
  • {{tab}} 

<html>
	<head>
		<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.1/angular.min.js"></script> <!-- Step-1 -->
		<script>
			angular.module('myApp', []); <!--Step-2 -->
			angular.module('myApp').controller('mobileController',function($scope){ <!-- Step-3 -->
				$scope.mobileData = [{'brand':'Apple','price':'800','model':'7s'},
				                     {'brand':'Samsung','price':'650','model':'SJ4'},
				                     {'brand':'Blackberry','price':'750','model':'Dtek'}];
				$scope.sortHeader='brand';
				$scope.sortTable=function(headerNam){
					if($scope.sortHeader!=headerNam){
						$scope.sortHeader=headerNam;
					}else{
						//already sorted so do reverse order
						$scope.sortHeader="-"+headerNam;
					}
				}
			});
		</script>
	</head>
	<body ng-app="myApp"><!-- Step-4 -->
		<div  ng-controller="mobileController"> <!-- Step-5 -->
			<h3>sortby: {{sortHeader}}</h3>
			<table>
      			<tr>
        			<th><a ng-click="sortTable('brand')">Brand</a></th>
        			<th><a ng-click="sortTable('price')">Price</a></th>
        			<th><a ng-click="sortTable('model')">Model</a></th>
      			</tr>
      				<tr ng-repeat="(index,mobile) in mobileData|orderBy:sortHeader"><!-- Step-6 -->
        				<td>{ {mobile.brand}}</td>
        				<td>{ {mobile.price}}</td>
        				<td>{ {mobile.model}}</td>
      				</tr>
    		</table>
		</div>
	</body>
</html>

Quidance

New concepts are below: goto All Steps for complete info
  • |orderBy:model variable used in dynamic sorting
  • Reverse sorting can be achieved by simply adding minus(-) sign with property name which is used in orderBy statement
  • orderBy:'-brand' prints records in reverse order by brand name
