ng-mousedown

The ngMousedown directive calls an event when mousedown happens.

Directive Info

This directive executes at priority level 0.

<button ng-mousedown="count = count + 1" ng-init="count=0"> Increment (when mouse is down) </button> count: { {count} }

Usage

as attribute: 
<ANY
  ng-mousedown="expression">
...
</ANY>
