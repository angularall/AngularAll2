Angular Directives

Directives extend HTML with custom attributes and elements with prefix ng-

The below are the most common directives needed for an AngularJS application.
  • ngApp The ngApp directive indicates the root element of the application and is mainly placed near the root element of the page - e.g. on the body or html tags.
  • ngBind The ngBind attribute binds the contents of model to a view. Whenever the content changes, the view gets updated.
  • ngInit The ngInit directive allows you to evaluate an expression in the current scope.
  • ngModel The ngModel directive binds an input,select, textarea etc to an application property.
  • ngController The ngController directive attaches a controller class to the view.
