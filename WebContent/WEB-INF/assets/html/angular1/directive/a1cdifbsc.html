ng-if

The ngIf directive removes or recreates a HTML element based on an expression.

If the expression assigned to ngIf evaluates to a false value then the element is removed, otherwise the element is added.

Directive Info

This directive creates new scope. This directive executes at priority level 600. This directive can be used as multiElement

Usage

<ANY
  ng-if="expression">
...
</ANY>
