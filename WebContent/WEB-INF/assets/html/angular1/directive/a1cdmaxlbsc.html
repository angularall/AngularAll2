ng-maxlength

The ngMaxlength adds the maxlength validator to ngModel. This is often used in Input text elements.

<input type="text" ng-model="model" id="input" name="input" ng-maxlength="maxlength" />

Directive Info

This directive executes at priority level 0.

Usage

as attribute: 
<ANY
  ng-maxlength="integer">
...
</ANY>
