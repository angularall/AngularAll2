ng-mouseup

The ngMouseup directive calls an event when mouseup happens.

Directive Info

This directive executes at priority level 0.

<button ng-mouseup="count = count + 1" ng-init="count=0"> Increment (when mouse is up) </button> count: { {count} }

Usage

as attribute: 
<ANY
  ng-mouseup="expression">
...
</ANY>
