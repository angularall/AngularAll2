ng-hide

The ngHide directive hides or hides the given HTML element

When the ngHide expression evaluates to a true value then the .ng-hide CSS class is added to the class attribute on the element causing it to become hidden. When false, the .ng-hide CSS class is removed from the element causing the element not to appear hidden.

Directive Info

This directive executes at priority level 0. This directive can be used as multiElement

Usage

as attribute: 
<ANY
  ng-hide="expression">
...
</ANY>
