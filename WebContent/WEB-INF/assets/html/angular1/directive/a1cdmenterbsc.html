ng-mouseenter

The ngMouseenter directive calls an event when mouseenter happens.

Directive Info

This directive executes at priority level 0.

<button ng-mouseenter="count = count + 1" ng-init="count=0"> Increment (when mouse is enter) </button> count: { {count} }

Usage

as attribute: 
<ANY
  ng-mouseenter="expression">
...
</ANY>
